## Projetos em destaque

### [RevOps Analytics Pipeline](https://github.com/joaoferro710/revops-analytics-pipeline)

Projeto end-to-end de dados para uma empresa SaaS B2B fictícia, cobrindo geração de dados sintéticos, ingestão em BigQuery, modelagem analítica com dbt, orquestração com Airflow e consumo final no Power BI.

**Tecnologias:** Python, BigQuery, dbt, Airflow, Docker Compose, Power BI

**Demonstra:**
- geração e ingestão de dados com Python
- uso do BigQuery como data warehouse analítico
- modelagem com dbt e validações de qualidade
- orquestração com Airflow
- análise de receita, churn, retenção e funil comercial

### [Macro Pulse](https://github.com/joaoferro710/Macro-Pulse)

Projeto de inteligência macroeconômica com ingestão de dados reais, detecção de anomalias, classificação de regimes macro e geração automatizada de briefings analíticos com LLM.

**Tecnologias:** Python, DuckDB, MotherDuck, LangChain, Groq, Streamlit

**Demonstra:**
- integração de APIs econômicas e de mercado
- persistência analítica com DuckDB e MotherDuck
- detecção estatística de anomalias em séries temporais
- uso de LLMs para geração de briefings
- construção de dashboard interativo com suporte multilíngue
